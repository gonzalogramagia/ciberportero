### 2) Way Too Long Words ([Link](https://codeforces.com/problemset/problem/71/A))

**Topic:** Strings and indexing

This problem introduced basic string manipulation: abbreviating long words by keeping the first and last character and replacing the middle with a number.

<Spoiler>
```python
n = int(input())
 
for _ in range(n):
    w = input().strip()
 
    if len(w) > 10:
        first_letter = w[0]
        letters_between = len(w[1:-1])
        last_letter = w[-1]
        print(first_letter + str(letters_between) + last_letter)
    else:
        print(w)
```

### 3) Team ([Link](https://codeforces.com/problemset/problem/231/A))

**Topic:** Conditionals and Counting

Given multiple team members‚Äô opinions (0 or 1), the task is to count how many problems the team will actually solve based on majority agreement.

<Spoiler>
```python
n = int(input())
 
solutions = 0
 
for _ in range(n):
    votes = list(map(int,input().split()))
    # input() ‚Üí "1 0 1" (a single string)
    # split() ‚Üí ["1", "0", "1"] (list of strings)
    # map(int, ...) ‚Üí 1, 0, 1 <iterator> producing integers
    # list(...) ‚Üí [1, 0, 1] (list of integers)
    consensus = 0
    for v in votes:
        if v:
            consensus += 1
    if consensus > 1:
        solutions += 1
 
print(solutions)
```
</Spoiler>

</ProblemSection>

### 4) Bit++ ([Link](https://codeforces.com/problemset/problem/282/A))

**Topic:** Simulation and String Parsing

A simple simulation problem where we need to increment or decrement a variable based on the input statement (e.g., "X++", "--X").

<Spoiler>
```python
n = int(input())	# 1 <= n <= 150
 
result = 0
 
for _ in range(n):
	character = input()
	match character[0]:
		case "X":
			if character[1] == "+":
				result += 1
			else:
				result -= 1
		case "+":
			result += 1
		case "-":
			result -= 1
 
print(result)
```
</Spoiler>

### 7) Beautiful Matrix ([Link](https://codeforces.com/problemset/problem/263/A))

**Topic:** Implementation

We are given a 5x5 matrix with a single '1' and need to calculate the minimum number of moves (swaps) to bring it to the center (3,3). The solution involves finding the coordinates of the '1' and calculating the Manhattan distance to the center.

<Spoiler>
```python
matrix = []
 
one_position = tuple()
 
swaps = 0
 
for row in range(5):
	matrix.append( list( map( int, input().split() ) ) )
	for column in range(5):
		if matrix[row][column] == 1:
			one_position = (column, row)
 
swaps = abs(one_position[0] - 2) + abs(one_position[1] - 2)
 
print(swaps)
```
</Spoiler>

</ProblemSection>

### 8) Helpful Maths ([Link](https://codeforces.com/problemset/problem/339/A))

**Topic:** Strings and Sorting

We need to reorder a sum of numbers (1, 2, 3) so they are in non-decreasing order. I solved this using a bubble sort algorithm, thanks to the [CS50 algorithms class](https://youtu.be/6Svu_ae5ebk).

<Spoiler>
```python
numbers = list( map( int, input().split("+") ) )
 
swaps = -1
 
while(swaps != 0):
	swaps = 0
	for i in range(len(numbers)-1):
		if numbers[i] > numbers[i+1]:
			aux = numbers[i]
			numbers[i] = numbers[i+1]
			numbers[i+1] = aux
			swaps += 1
 
result = "+".join(list(map(str,numbers)))
 
print(result)
```
</Spoiler>

</ProblemSection>

### 9) Soldier and Bananas ([Link](https://codeforces.com/problemset/problem/546/A))

**Topic:** Math

A soldier wants to buy bananas where the cost increases for each banana. We need to calculate the total cost using an arithmetic series formula (or a loop) and determine how much money he needs to borrow.

<Spoiler>
```python
cost, dollars, bananas = map( int, input().split() )
 
total = 0
borrow = 0
 
for i in range(1, bananas + 1):
	total += cost * i
 
if(total > dollars):
	borrow = total - dollars
 
print(borrow)
```
</Spoiler>
